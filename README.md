# Account Dashboard Bank Buttons

Module Odoo 18 Community ajoutant deux boutons sur les journaux bancaires du dashboard :

- Relevés bancaires
- Import CAMT

## Dépendances

- account
- account_statement_import_file (OCA)

## Installation

1. Copier le module dans votre dossier custom addons
2. Redémarrer Odoo
3. Mettre à jour la liste des applications
4. Installer "Account Dashboard Bank Buttons"
